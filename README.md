# How to start

1. Install MeteorJS platform
  - Linux: `curl https://install.meteor.com/ | sh`
  - Windows: https://install.meteor.com/windows

2. Start MeteorJS application from the `app` directory:
`meteor --settings=config/dev/settings.json`
